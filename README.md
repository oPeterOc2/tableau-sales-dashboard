# 📊 Tableau Sales Dashboard (Superstore Dataset)

This project is a **basic Tableau dashboard** built using the official Tableau *Sample – Superstore* dataset.  
It demonstrates essential Tableau skills including data connection, chart creation, filters, and dashboard design.  
The goal is to showcase foundational data‑visualization ability for portfolio and interview purposes.

## 🔧 Features

### **1. Sales by Region (Bar Chart)**
A bar chart showing total sales across the four regions:
- East  
- West  
- Central  
- South  

### **2. Sales Trend Over Time (Line Chart)**
A line chart visualizing sales performance over time using the `Order Date` field.

### **3. Region Filter**
Interactive filter allowing users to select a region and update the entire dashboard accordingly.

### **4. Interactive Dashboard**
Both charts are combined into a single dashboard.  
Selecting a bar in the bar chart automatically filters the line chart.
